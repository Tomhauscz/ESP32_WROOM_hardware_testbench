# ESP32_WROOM_hardware_testbench
Testbench project for ESP32-WROOM microcontroller and various peripheral devices. Used devices are VL53L0X TOF distance sensor, ST7796/7789 TFT displays, PCA9685 16-channel PWM servo motor driver, rotary encoder and push buttons.

## Connected deviced list
| Bus/Peripheral  | Connected device |
| --- | --- |
| I2C | VL53L0X TOF distance sensor |
| I2C | PCA9685 16-channel PWM driver for stepper motors |
| SPI | 3.5" TFT Display 320x480 (ST7796U driver) |
| GPIO | Rotary Encoder KX-040 with built-in push button |
| GPIO | Push buttons |
